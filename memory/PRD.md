# Оценка PRO - Приложение для оценки ущерба автомобиля

## Оригинальное ТЗ
Веб-приложение для оценки ущерба автомобиля, адаптированное под мобильные экраны, с использованием Django и HTML шаблонами.

## Архитектура
- **Backend**: Django 5.2 с ASGI через uvicorn
- **Frontend**: Django HTML Templates + Tailwind CSS (CDN)
- **База данных**: SQLite
- **Proxy**: React dev server проксирует на Django (порт 8001)

## User Personas
1. **Владелец авто** - отправляет заявку на оценку ущерба после ДТП
2. **Администратор** - обрабатывает заявки через Django Admin
